# JSON Examples

* **Complete first** yang/yang-library.json with the required configuration to validate the examples
  + "yang-library.json" is structured following [RFC7895](https://datatracker.ietf.org/doc/html/rfc7895)    
* Examples that need to be validated should be **prefixed with "valid-"**.
* Other examples are not validated, as these may be crafted to illustrate malformed messages, etc.
